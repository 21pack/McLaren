# McLaren

Modern C++ Library for Axonometric Rendering Engine

![demo](/demo.gif)

## Usage

### Install

If you use Linux, install SFML's dependencies using your system package manager. On Ubuntu and other Debian-based distributions you can use the following commands:

```shell
sudo apt update
sudo apt install \
    libxrandr-dev \
    libxcursor-dev \
    libxi-dev \
    libudev-dev \
    libfreetype-dev \
    libflac-dev \
    libvorbis-dev \
    libgl1-mesa-dev \
    libegl1-mesa-dev \
    libfreetype-dev
```

### Build

```shell
./scripts/build.sh
```

### Play

```shell
./scripts/play.sh
```

### Test

```shell
./scripts/test.sh
```
